Das Mikrofon deines Handys in einen Browser streamen (über WLAN)

Möchtest du das Mikrofon deines Handys in einen Browser streamen – im selben Wi-Fi? RemoteMic macht ein Android-Handy zu einem Live-Mikrofon, das jeder Computer, jedes Tablet oder ein anderes Handy einfach über einen Weblink abspielen kann — keine App für die Zuhörer, keine Kabel und kein Cloud-Konto. Handy ausrichten, Seite öffnen, und deine Stimme ist direkt im Browser.

Was du brauchst

  • Ein Android-Handy (Android 11 oder neuer) mit installiertem RemoteMic — das ist das Mikrofon.
  • Ein beliebiges Gerät mit modernem Browser — Chrome, Safari, Edge oder Firefox — im selben Wi-Fi.

Schritt für Schritt

  1. Mikrofon in RemoteMic starten. Das Handy nimmt Audio auf und stellt einen lokalen Stream in deinem Netzwerk bereit.
  2. Zuhör-Link finden. Die App zeigt eine Webadresse (und einen QR-Code) für den Stream — eine Adresse in deinem lokalen Netzwerk, die du in jedem Browser öffnen kannst.
  3. Auf dem Computer öffnen. Gib den Link im Browser deines Laptops oder Desktop-PCs ein, oder scanne den QR-Code mit einem anderen Handy. Die Wiedergabe startet direkt auf der Seite.
  4. Sprechen. Deine Stimme wird in Echtzeit über die Lautsprecher oder Kopfhörer des Computers wiedergegeben. Passe die Verstärkung am Handy an, wenn du mehr oder weniger Pegel brauchst.

Müssen Zuhörer etwas installieren?

Nein. Genau darum geht es — die Zuhörseite ist nur eine Webseite. Alle, mit denen du den Link teilst, können im selben Netzwerk direkt im Browser zuhören, ohne etwas herunterzuladen. Nur auf dem Mikrofon-Handy läuft RemoteMic.

Können mehrere Geräte gleichzeitig zuhören?

Ja. Teile denselben Link, und jeder Browser im Netzwerk kann den Stream gleichzeitig abspielen — zum Beispiel ein Laptop, ein Desktop-PC und ein paar Handys zusammen. Der Mechanismus ist derselbe, egal ob eine Person zuhört oder ein ganzer Raum.

Wie groß ist die Verzögerung?

Bei normalem Wi-Fi oder Hotspot ist die Verzögerung meist niedrig genug für Erzählungen, Sprachmonitoring, Hörunterstützung oder um Audio in eine Aufnahme oder einen Anruf einzuspeisen. Typische Setups mit Mittelklasse-Handy und Router können je nach Wi-Fi-Bedingungen und Audiopufferung bei etwa 150 ms liegen. Nicht gedacht für exakte Lippensynchronität mit Video.

Kein Wi-Fi-Netzwerk? Nutze den integrierten Hotspot

Wenn kein gemeinsames Netzwerk verfügbar ist, kann RemoteMic einen eigenen privaten Hotspot erstellen. Verbinde den Computer mit diesem Hotspot, öffne den Link, und du streamst ohne Internet und ganz ohne Router.

Praktisch für

  • Handy als PC-Mikrofon nutzen — über Wi-Fi statt mit einem USB-Mikrofon.
  • Monitoring — höre einen Raum von einem anderen Gerät aus, während du alles einrichtest.
  • Gruppen-Zuhören — viele Browser hören einen Stream, ohne zusätzliche Geräte.

Sieh dir an, wie RemoteMic funktioniert, oder lies den Leitfaden zum Betrieb eines drahtlosen Tour-Guide-Systems, wenn du zu den Handys einer ganzen Gruppe streamen möchtest.

Häufige Fragen

Muss ich auf dem Computer etwas installieren?

Nein. Das zuhörende Gerät öffnet einfach eine Webadresse in einem modernen Browser — Chrome, Safari, Edge oder Firefox. Nur das Handy benötigt RemoteMic.

Wie öffne ich den Stream?

RemoteMic zeigt einen Link und einen QR-Code an. Gib den Link in einen Browser im selben WLAN ein oder scanne den QR-Code, und die Wiedergabe startet.

Können mehrere Computer und Handys gemeinsam zuhören?

Ja. Teile denselben Link; jeder Browser im Netzwerk kann den Stream gleichzeitig abspielen.

Wie groß ist die Verzögerung?

In einem lokalen Netzwerk meist deutlich unter einer Sekunde — passend für Erläuterungen, Monitoring und Hörunterstützung.

Wird Internet benötigt?

Nein. Es funktioniert in jedem lokalen WLAN und kann einen eigenen Hotspot bereitstellen, wenn gar kein Netzwerk vorhanden ist.
